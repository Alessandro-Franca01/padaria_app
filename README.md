# Padaria App

Padaria App é um aplicativo Flutter que oferece uma experiência de compra online para padarias, permitindo que os usuários explorem uma variedade de produtos de panificação, façam pedidos e aproveitem promoções exclusivas.

## Funcionalidades

- **Navegação Intuitiva**: Explore produtos e categorias com facilidade.
- **Promoções Exclusivas**: Descubra ofertas especiais disponíveis apenas no aplicativo.
- **Carrinho de Compras**: Adicione produtos ao carrinho e finalize suas compras de forma prática.
- **Gerenciamento de Conta**: Acesse e edite suas informações pessoais.
- **Autenticação Segura**: Login e registro de usuários com segurança.

## Getting Started

Estas instruções ajudarão você a configurar e executar o projeto localmente para desenvolvimento e testes.

### Pré-requisitos

- Flutter SDK: [Instale o Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK: Incluído com o Flutter
- Editor de Código: Recomendamos o [Visual Studio Code](https://code.visualstudio.com/) com a extensão Flutter

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/padaria_app.git
   cd padaria_app
   ```

2. Instale as dependências:
   ```bash
   flutter pub get
   ```

3. Execute o aplicativo:
   ```bash
   flutter run
   ```
